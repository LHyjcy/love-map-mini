# PRIVACY — love-map-mini

本应用涉及情侣关系数据、照片和位置，采用**隐私优先**的默认设置。

## 核心原则

- **不做后台持续定位**。位置仅在用户主动打卡或开启临时共享时采集。
- **位置共享默认关闭**。伴侣只有在用户明确选择分享后才能看到所选位置。
- **打卡仅由用户主动发起**，且临时共享可设置过期时间。
- **公开地图默认关闭**，且不默认暴露精确的家、学校、工作坐标。
- 用户可删除打卡、回忆和照片元数据。
- 每个私有接口都必须校验 `userId` 与 `coupleId` 访问权限。

## 数据删除

用户可删除：打卡（checkin）、回忆（memory）、照片元数据（media）。删除采用软删除，
后续可提供彻底清除入口。

## 敏感地点

建议用户不要把家、学校、工作等敏感地点设为公开可见。公开地图坐标默认做轻微模糊
（保留 3 位小数）。
