## 0. 准备与启动

- [ ] **启动后端**：双击 `E:\dt\scripts\start-dev.bat`（或 PowerShell 跑 `scripts\start-dev.ps1`），保持窗口开着。
  - 看到 `listening on http://0.0.0.0:3000`；浏览器访问 `http://localhost:3000/health` 返回 `{"success":true,...}`。
  - MySQL 已是开机自启服务，无需手动起；脚本会自动确认。
- [ ] **导入小程序**：微信开发者工具 → 导入项目 → 目录 `E:\dt\apps\miniprogram`（AppID 可用「测试号」）。
- [ ] **关闭域名校验**：右上「详情 → 本地设置」勾选 **「不校验合法域名、web-view、TLS、HTTPS」**（本地 `http://localhost:3000` 必需）。
- [ ] `app.js` 的 `globalData.baseUrl` = `http://localhost:3000`（默认即是）。
- [ ] 编译无报错；底部 tabBar 为 **首页 / 地图 / 计划 / 互动 / 我的**。

> 开发态已知限制（非 bug）：① 微信登录返回 501（未配 AppID）；② 照片本地 provider 不真正上云，封面显示暖色占位；③ AI 文案走本地模板（`source:template`，未配 AI key）；④ 订阅消息为占位（模板 ID 待填）；⑤ 位置共享 WS 无 Redis 走内存兜底。

## 3. 地图（地图首页 + 真实地图 + 足迹地图 + 位置共享）

### 3.1 地图首页（tab）
- [ ] 顶部四项足迹统计：点亮省 / 城市 / 地点 / 回忆数。
- [ ] 三入口：真实地图 / 足迹地图 / 位置共享，跳转正确；下拉刷新。

### 3.2 真实地图
- [ ] 原生 map 渲染，marker 来自地点/回忆/打卡。
- [ ] **点击 marker → 立体底部卡**：升起动画 + 封面照片（无图为占位）+ 最新回忆标题/文案；「查看详情」进详情。
- [ ] 右上「⤢ 地图大小」循环切换地图高度；双指可缩放。
- [ ] 悬浮入口：＋地点 / 打卡 / 回忆 / 共享。

### 3.3 足迹地图（立体相册重点）
- [ ] 全国页 canvas 绘制省份，已去过省份**点亮**；统计文案正确。
- [ ] **点击点亮省份 → 弹立体预览卡**（省名 + 查看城市足迹）；点未点亮省提示「还没点亮」。
- [ ] 进省份页：canvas 城市点亮；点击点亮城市弹预览卡（地点/回忆数）→ 进城市页。
- [ ] **城市页 = 立体回忆相册**：横向 swiper 景深卡片（居中放大、两侧缩小）；封面大图压标题、2 行文案、标签、日期心情；多图角标；点封面预览大图；点卡片进回忆详情。
- [ ] **验收①**：在杭州加一个 visited 地点（城市填「杭州」）→ 回足迹首页，浙江/杭州被点亮、统计 +1。

### 3.4 位置共享（隐私重点）
- [ ] 未开启时：TA 看不到我的位置；页面有隐私文案。
- [ ] **开启共享 30 分钟 / 2 小时**：触发定位授权（授权框文案含「仅前台、可随时关闭」）；地图显示「我」marker。
- [ ] 双人：A 开启并移动后，B 在该页看到 A 的「TA」marker + 最近更新时间 + 直线距离。
- [ ] **停止共享**：TA 立即不可见。
- [ ] 拒绝定位权限：友好引导「去设置」，不崩溃。
- [ ] 离开页面停止前台定位（不后台持续定位）。

## 12. 联调常见问题

| 现象 | 原因/处理 |
|---|---|
| 请求 `ERR_CONNECTION_REFUSED` | 后端没起 → 跑 `scripts\start-dev.bat` 并保持窗口 |
| 请求被拦 / 域名不合法 | DevTools 未勾「不校验合法域名」 |
| 微信登录 501 | 未配 `WECHAT_APP_ID/SECRET`（开发态预期，用演示/体验登录） |
| 照片不持久 / 封面空 | 本地 `STORAGE_PROVIDER=local`，未配 COS（占位渲染正常） |
| AI 文案是模板 | 未配 `AI_API_KEY`（`source:template`，属预期） |
| 中文存成 `??` | 仅命令行测试编码问题；小程序 `wx.request` 默认 UTF-8，不受影响 |
