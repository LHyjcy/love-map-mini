# PRD — love-map-mini

## 产品定位

情侣共同记录去过的地方、当时的故事和照片，同时用任务、积分、纪念日、打卡增强日常互动。
一个**私密的**情侣双人小程序，不是陌生人社交或公开社区。

## 核心闭环

1. **地图回忆闭环**：一起去某地 → 打卡 → 上传照片 → 写故事 → 地图点亮 → 周年回顾。
2. **任务互动闭环**：发布任务 → 对方完成 → 确认 → 获得积分 → 兑换情侣券 → 生成新回忆。
3. **约会计划闭环**：添加想去地点 → 设为计划 → 到达打卡 → 变成共同回忆 → 路线沉淀。

## MVP 范围

第一版必须做：微信登录、情侣绑定、地图地点、回忆故事、图片元数据、主动打卡、任务、积分、商城、日程、隐私设置。

第一版暂不做：后台持续定位、陌生人社交、公开社区、复杂支付、AI 生成日记、多情侣关系、实时轨迹回放。

## 功能模块

Auth、Couple、Place、Memory、Media、Checkin、Task、Point、Shop、Event、Privacy、PublicShare。

## 页面（小程序）

| 页面 | 说明 |
|---|---|
| home | 首页 dashboard：在一起天数、积分、最近回忆/任务/日程、快捷入口 |
| map | 地图：地点 marker、点击查看回忆卡片、添加地点 |
| tasks | 任务列表与状态流 |
| shop | 情侣商城与兑换 |
| events | 日程、纪念日、倒计时 |
| me | 我的、隐私设置、公开地图开关 |

> 详细页面 / 接口随各 Phase 推进逐步补全。Phase 1 仅建立骨架。
